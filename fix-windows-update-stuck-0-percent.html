Windows Update Stuck at 0 Percent (Fix It Now)

Updated Feb 13, 2026

⚠️ Immediate Fix

Update stuck at 0% or "Pending"? The update service has hung. You must restart the service. Open "Services.msc", right-click "Windows Update", and select "Restart". If it stays stuck, you need to delete the specific cache folder `C:\Windows\SoftwareDistribution`.

Why This Happens

  • Corrupt Cache: Partial download files.
  • Service Hang: Update agent not responding.
  • Disk Space: Not enough room to unpack.

Step-by-Step Fix

  1. Stop Services: Open CMD as Admin. Type `net stop wuauserv` and `net stop bits`.
  2. Delete Cache: Navigate to `C:\Windows\SoftwareDistribution`. Delete EVERYTHING in this folder. (It is safe, these are just temp download files).
  3. Restart Services: In CMD, type `net start wuauserv` and `net start bits`.
  4. Check Again: Go back to Settings > Update & Security > Check for Updates. It will re-download fresh.

When to Stop

If updates essentially never work, your OS might be too damaged. Use the "Windows Media Creation Tool" to perform an "In-Place Upgrade". This reinstalls Windows over itself, keeping your files but replacing all system files with fresh ones.
