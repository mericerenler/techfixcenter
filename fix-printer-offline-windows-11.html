Fix Printer Offline in Windows 11

Updated Feb 13, 2026
Advertisement

The Problem: You try to print a document, but nothing happens. In Settings, your printer status is "Offline."

The Short Answer: The Print Spooler service might have crashed, or Windows has set the printer to "Use Printer Offline" mode by mistake.

Quick Fixes

  • Method 1: Restart Print Spooler
  • Method 2: Check 'Use Printer Offline' Setting
  • Method 3: Re-add Printer

Method 1: Restart Print Spooler

This service manages all print jobs. Restarting it clears stuck files.

  1. Press Windows + R type services.msc.
  2. Find Print Spooler.
  3. Right-click > Restart.
Advertisement

Method 2: Check 'Use Printer Offline' Setting

Sometimes this setting gets checked accidentally.

  1. Go to Settings > Bluetooth & devices > Printers & scanners.
  2. Click your printer.
  3. Click Open print queue.
  4. In the menu bar, click Printer.
  5. Uncheck Use Printer Offline if it is checked.

Method 3: Re-add Printer

If it is still stuck, remove it and let Windows find it again.

  • Go to Printers & scanners settings.
  • Click Remove next to your printer.
  • Restart PC.
  • Click Add device to reinstall.
