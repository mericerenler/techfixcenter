Fix Windows Update Error 0x80070002

Updated Feb 13, 2026
Advertisement

The Problem: You start a Windows Update, but it stops immediately with code 0x80070002.

The Short Answer: This specific code means "The system cannot find the file specified." Essentially, Windows Update listed a file to download, but it appeared to be missing or mismatched.

Quick Fixes

  • Method 1: Stop the Windows Update Service
  • Method 2: Delete DataStore Files
  • Method 3: Restart and Check

Method 1: Stop the Windows Update Service

To safely delete the corrupt files causing this error, we first need to stop the update service.

  1. Press Windows Key + R to open Run.
  2. Type services.msc and press Enter.
  3. Scroll down to find Windows Update.
  4. Right-click it and select Stop.
Advertisement

Method 2: Delete DataStore Files

Now we remove the files that are confusing Windows.

  1. Open File Explorer and go to C:\Windows\SoftwareDistribution\DataStore.
  2. Delete all files and folders inside this folder.
  3. Go back to C:\Windows\SoftwareDistribution\Download.
  4. Delete everything in there too.

Method 3: Restart Services

Now turn it back on.

  • Go back to the Services window.
  • Right-click Windows Update and select Start.
  • Try running Windows Update again. represent.
