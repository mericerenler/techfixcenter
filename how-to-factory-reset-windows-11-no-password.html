Factory Reset Windows 11 Without Password

Updated Feb 13, 2026
Advertisement

The Problem: You are locked out of your PC. Maybe you forgot the password or bought it used. You cannot get into Settings to reset it.

The Short Answer: You do not need the password to reset the machine. You can trigger the "Reset this PC" command directly from the lock screen using a keyboard shortcut.

Steps

  • Step 1: Access Recovery Environment
  • Step 2: Start the Reset
  • Step 3: Cloud vs Local Install

Step 1: Access Recovery Environment

CRITICAL TIMING Hold the Shift key down. Keep holding it while you click Power > Restart. Do not release Shift until the screen turns blue.
  1. At the login screen, click the Power icon (bottom right).
  2. Hold down the Shift key.
  3. Click Restart.
  4. Release Shift only when you see "Choose an option".
Advertisement

Step 2: Start the Reset

You are now in the Windows Recovery Environment (WinRE).

  1. Click Troubleshoot.
  2. Click Reset this PC.
IMPORTANT NOTE Choose Remove everything. If you choose "Keep my files", Windows will ask for your password (which you don't know), and you will be stuck. "Remove everything" bypasses the password check.

Step 3: Cloud vs Local Install

Windows will ask how to reinstall the system.

PRO TIP Choose Local Reinstall if your internet is slowâ€”it's much faster. Only choose Cloud Download if the reset fails.
  • Local Reinstall: Uses existing files. Fast.
  • Cloud Download: Downloads fresh Windows. Slower.

Confirm the reset. Your PC will restart several times. This takes about 1-2 hours.

Advertisement

Still Not Fixed?

If the reset fails, you will need a USB drive with Windows 11 installation media to reinstall from scratch.
