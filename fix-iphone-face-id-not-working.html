iPhone Face ID Not Available (Fix It Now)

Updated Feb 13, 2026

⚠️ Immediate Fix

Is Face ID failing or showing "Not Available"? The TrueDepth camera is likely blocked or confused. Clean the top notch area thoroughly with a cloth. If it persists, Reset Face ID in settings—software glitches often disable the biometric scanner for security.

Advertisement

Why This Happens

  • Smudge on Sensor: Oil/dirt blocks the IR dot projector.
  • iOS Security Lockout: Too many failed attempts disabled it.
  • Screen Protector: It may be peeling and covering the camera.

Step-by-Step Fix

  1. Clean the Notch: Use a microfiber cloth to wipe the top sensor array. Even a tiny smudge can block the IR dots.
  2. Reset Face ID: Go to Settings > Face ID & Passcode. Tap "Reset Face ID". This deletes the old mathematical model and lets you scan again.
  3. Check for Updates: Go to General > Software Update. Apple frequently patches Face ID bugs in minor iOS updates.
  4. Reset All Settings: If still broken, go to General > Transfer or Reset > Reset > Reset All Settings. This fixes system permissions without deleting data.

When to Stop

If you see a warning saying "A problem was detected with the TrueDepth Camera", your hardware is broken. This often happens after water exposure or a hard drop. The sensor cannot be repaired, only the whole unit replaced by Apple.
