Windows Can't Reset This PC (Fix It Now)

Updated Feb 13, 2026

⚠️ Immediate Fix

Reset failing at 30% or showing "There was a problem resetting your PC"? Your recovery image is corrupt. You must run the system file checker. Open Command Prompt as Administrator and type `sfc /scannow`. If that fails, using a USB Installation Media is the only way to bypass the corruption.

Why This Happens

  • Corrupt WinRE: Recovery partition is damaged.
  • Disk Errors: Bad sectors preventing file write.
  • Virus: Malware blocking the wipe.

Step-by-Step Fix

  1. Run SFC Scan: Search "CMD", right-click "Run as Admin". Type `sfc /scannow`. Let it finish. It usually repairs the recovery agent.
  2. Disable ReAgentC: In CMD, type `reagentc /disable` then `reagentc /enable`. This resets the recovery environment link.
  3. Check Disk: Type `chkdsk /f /r c:` and restart. Let it repair bad sectors.
  4. USB Clean Install: If reset fails, download the "Windows Media Creation Tool" on another PC. Create a bootable USB. Boot from it and select "Custom: Install Windows Only" to format the drive manually.

If you cannot even install fresh from USB (errors during install), your hard drive or RAM is defective. Try removing one stick of RAM and trying again.
